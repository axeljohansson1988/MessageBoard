SETUP API
Prerequisites
- .NET runtime installed

Instructions to run API
- Open a terminal or cmd
- Navigate to folder where MessageBoard.sln is located
- Run command dotnet run
- Launch a browser at localhost:5000/swagger to examine the endpoints of the API

Instructions to run unit tests
- Open the solution in Visual Studio
- Right-click project MessageBoard.API.Tests
- Click Run Tests


SETUP REACT APP
Prerequisites
 -Node.js installed

Instructions to run app
- Navigate to folder messageboardapp
- Run command npm install
- Run command npm start
- Launch the app in a browser at localhost:3000


Start messaging! :)
